Save to GitHub

To have panel changes saved to GitHub (not just in your browser), set up the token in Netlify:

  1. In Netlify, open your site.
  2. Go to Site settings → Environment variables.
  3. Click Add a variable or Add from .env.
  4. Name: GITHUB_TOKEN
  5. Value: your GitHub token (created in GitHub → Settings → Developer settings → Personal access tokens, with repo permission).
  6. Save and run a new Deploy (Deploys → Trigger deploy → Clear cache and deploy site).

After setting the token, panel saves will be pushed to GitHub and visitors will see changes in 1–2 minutes.

If your repo has a different name or owner, edit netlify/functions/update-site-data.js (variables owner and repo).
